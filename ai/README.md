学习课程 

1. 李宏毅：[生成式人工智慧與機器學習導論2025](https://www.youtube.com/playlist?list=PLJV_el3uVTsMMGi5kbnKP5DrDHZpTX0jT) （目前只4个视频，还在更新中），文理工科都能看懂，推荐所有人看
2. [生成式AI時代下的機器學習(2025)](https://www.youtube.com/playlist?list=PLJV_el3uVTsNZEFAdQsDeOdzAaHTca2Gi) 13讲，也推荐看
3. [生成式AI導論 2024](https://www.youtube.com/playlist?list=PLJV_el3uVTsPz6CTopeRp2L2t4aL_KgiI) 20讲，上述课程的2024版本
